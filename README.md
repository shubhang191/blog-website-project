# My JS Blog App

This is a simple CRUD application built with Vanilla JavaScript.

**Features:**
- Create new blog posts with images.
- Data is persisted using Local Storage.
- Delete posts.

**Note to Reviewers:**
- Since this uses Local Storage, the blog will appear empty when you first visit the link. Please try adding a post to see it in action
